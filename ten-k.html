Ten Thousand Books - Increasing Outreach for Diversity With Literature

UX/UI Case Study

Overview

Ten Thousand Books Is a Non-Profit organization that aims to add literature with diverse voices and experiences to school classrooms in Minnesota and Beyond. They have a website thats fun looking and aimed towards kids but seems to be lacking in several usability areas. I was able to work with a group and assess the tasks at hand.

The Problem

Ten Thousand Books is struggling to increase outreach to teachers and students. Their presence on social media is acceptable, but without a clear central hub for more information on their organization, the user could lose interest on the way. In terms of design, the current website features tacky colors and odd placements for links. Fixing those small things could make a huge difference in building a better online experience for those interested, as well as making some needed additions.

Nielson Norman Heuristic Analysis of Current Site

  1. Visibility of system status
    • Navigation is working and shows active states for each main page. Although When a sub category is selected it still shows the main page as active, not the sub category.
  2. Match between system and the real world
    • Website seems to exhibit clear terminology
    • There is no use of icons to return or see what your interacting with, other than the active state of the page the user is on.
  3. User control and freedom
    • Main logo is easily accessible from every page and links back to home on click. Since there is little amounts of information on each page it's not a priority for a secondary control system to be present.
  4. Consistency and standards
    • Each page matches it's intended destination. However, all body designs seem to never stay consistent. Pictures, text blocks, and buttons are not consistent across all pages.
  5. Error prevention
    • Only example of this sits on the contact page. If a user Misses filling out an email, the message wont send and will prompt user that the field must be filled out.
    • Another problem is that if a user only fills out the email field and presses send. The message will send and prompt the user with a "We will be in touch!" pop up. There should be a form of error prevention when a user does not fill out the name, phone, and message field as well.
  6. Recognition rather than recall
    • Site is very informational and there is not any content that fits in the criteria of this standard
  7. Flexibility and efficiency of use
    • Using tab to sort through text fields. There are no text accesibility features though.
  8. Aesthetic and minimalist design
    • Distracting colors and non consistent design is a common theme to this site. That shouldn't feel like the main issue but it defintly feels like it. It's because the colors are not directly putting focus on UI elements. The website is simple. The colors should be simple as well and they do not match attention getting standards for the UI.
  9. Help users recognize, diagnose, and recover from errors
    • Only has error prevention with pop ups in the contact section. It's hard to go in the wrong direction as a user for this site.
  10. Help and documentation
    • No help documentation provided from website.

UI Analysis

  1. 1.1 Navigation
    • Navigation is short and direct but it is missing other clickable features within the page that could be added to the top nav.
  2. 1.2 Margins
    • Semi-passes accessibility (not comepletly accessible though) but color is an odd choice for margins.
  3. 1.3 Sub Navigation
    • Seems out of place and could be added to the top navigation. For example, Donate is in 2 locations being displayed on the same page
  4. 1.4 Card
    • Having a card like this is fine. However, a suggestion would be to keep the colors consistent. Example being bright red on dark red.
  5. 1.5 Card Grid
    • This is an aspect that can be kept. Good use of space. This grid type could be used in the curated books section that we would plan to introduce.
  6. 1.6 Footer Navigation
    • This is an aspect that can be kept. Good use of space. However, section could be an option in the navigation.
  7. 2.1 Contact form
    • A tad bit close together. No margins. Gets the point across though. The submit button text has no padding and seems large.
  8. 3.1 Carousel of curated books
    • Good idea. However it moves slowly and UI for the carousel is visually bland/cant really see it.

Phase 3: Ideation And Prototyping Phase

Site Map And User Flow

The Existing site doesn’t have a lot of content within the tabs and didn’t need to be seperated. Only additions or certain changes will help make the user experience more clear. As a group, the site map was easy to plan and create.

I was able to create a User Flow with scenarios that were important processes. The three being the volunteer, donation, and contact processes.

Lo-Fi Prototype

×

Adding a Blog was a client request. Ten Thousand Books would like to update users on what they are doing or talking about. This section would make users want to return, kind of like a news source.

Separating “About Us” from “Our Work” was a needed decision. The Existing site only shows what schools they work with, but no information on what they accomplished. Users being able to see accomplishments from the non-profit will give them incentive to possibly get involved.

Donate and Volunteer were to moved to the top right next to the search bar. During research, we noticed that users were slow to go to donate when it was within the main nav. Separating it from the main nav and keeping it by a place where users go to search for information (like a search bar), gives it hierarchal importance. When building the Hi-Fi Prototype, It would be smart to give it a different button color.

Phase 5: Final Prototype

How Do Users Feel About The Change?

We asked people to complete simple tasks while using the redesign prototype to see what patterns and opinions they provide

  • Find information on the backstory of 10K books
  • Figure out what this non profit aims to accomplish
  • Find what books 10K books donates to schools
  • Donate to 10K books

Common Observations:

  • Users tended to either find information on the non profit within the homepage or in the tab “About Us”.
  • The “Books” tab and content layout makes it seem like a recommended section instead of what 10K books would curate to schools.
  • “The Donation being at the top is way more noticable now” - (Returning test subject)
